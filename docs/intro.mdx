## Creating a new project from Boilerplate ©️

To create a new project using the **Boilerplate**, follow the steps below:

- Clone the **Github Repository** of the Boilerplate by running the command below:

```bash
git clone https://github.com/Cavin6080/bloc_boilerplate.git
```
:::warning
This repository is private as of now so in case of any issue. You can reach out to 
[Cavin](https://www.linkedin.com/in/cavin-macwan/)
:::

You can type this command into **Command Prompt**, **Powershell**, **Terminal**, or any other **integrated terminal** of your code editor.

- Run the below command to install all necessary dependencies you need to run your project:

```bash
flutter pub get
```

## Launch your App

Launch your app by either through your IDE or by running the command below:

```bash
flutter run -t lib/main_development.dart
```

- `flutter pub get`:  Installs the neccessary dependencies for your project ⬇️

- `flutter run -t lib/main_development.dart`:  Builds your flutter app and launches that on
either emulator or your device 🚀
